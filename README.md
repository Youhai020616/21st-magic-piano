# 21st Magic Piano

一个基于 React 和 Web Audio API 的交互式钢琴游戏。

## 特性

- 🎹 完整的钢琴键盘布局
- 🎵 真实的钢琴音效
- ⌨️ 键盘控制支持
- 🎮 练习和挑战模式
- 📱 响应式设计

## 技术栈

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion

## 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 键盘映射

### 低音区
- 白键：1 2 3 4 5 6 7
- 黑键：Q W R T Y

### 中音区
- 白键：8 9 0 A S D F
- 黑键：I O Z X C

### 高音区
- 白键：G H J K L ; /
- 黑键：B N M , .
